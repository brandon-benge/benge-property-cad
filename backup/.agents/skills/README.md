# Skills

Project-specific skills are mapped one-to-one with the domain agents.

All skills assume the agent using them is competent with FreeCAD and Python, including the FreeCAD document/object model, `Part` geometry, placements, recompute/save behavior, Python module reloading, and the project-owned `model.py` / `config.py` workflow.

Current skills:

- `structure-review/` maps to `../agents/structure-review.md`
- `wiring-review/` maps to `../agents/wiring-review.md`
- `plumbing-review/` maps to `../agents/plumbing-review.md`
- `site-layout-review/` maps to `../agents/site-layout-review.md`
- `buildability-review/` maps to `../agents/buildability-review.md`
- `architect-review/` maps to `../agents/architect-review.md`

Each skill must keep recommendations scoped to project-owned files unless the user is explicitly changing upstream tooling.
